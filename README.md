# Unity-AI-Chat-App
AI-powered chat system built in Unity with real-time API integration and interactive UI.

---

## 🚀 Features

- 💬 Chat UI (User & AI bubbles)
- 🤔 Thinking animation (AI processing state)
- ⌨️ Typing effect (real-time response feel)
- 🌐 API integration (Python backend / local LLM)
- 📱 Mobile support (Android-ready)

---

## 🛠️ Tech Stack

- Unity (C#)
- TextMeshPro (UI)
- Python (Backend API)
- LLM Integration (Local / API-based)

---

## 🎮 How It Works

1. User enters a message  
2. Unity sends request to backend API  
3. AI processes input  
4. Response is displayed with typing animation
